#### 이미지 파일(.ico, .jpg, .png)로 추가하기

`/app` 디렉터리에 `favicon`, `icon` 또는 `apple-icon` 이미지 파일을 배치하여 이미지 파일을 사용하여 앱 아이콘을 설정할 수 있다. 파비콘 이미지는 `app/`의 최상위 수준에만 위치할 수 있다.

Next.js는 파일을 평가하고 앱의 `<head>` 요소에 적절한 태그를 자동으로 추가한다.

| File convention | Supported file types          | Valid locations |
| --------------- | ----------------------------- | --------------- |
| favicon         | .ico                          | app/            |
| icon            | .ico, .jpg, .jpeg, .png, .svg | app/\*_/_       |
| apple-icon      | .jpg, .jpeg, .png             | app/\*_/_       |

#### 구성 방법

루트 `/app` 경로 세그먼트에 favicon.ico 이미지 파일을 추가한다.

```js
// <head> output
<link rel="icon" href="/favicon.ico" sizes="any" />
```

여기까지 공식 문서를 통해 작성한 내용이다. 이제 한번 따라해보자!

1. 이미 생성된 `public`폴더 안에 파비콘으로 설정할 이미지를 추가한다.
   ![image](https://github.com/tamoimi/tami-dev-blog/assets/100749520/433f7374-4756-4073-afb6-c0c29a0b7b51)

2. `<head>` 태그가 위치한 `layout.tsx`에 아래와 같이 `link` 태그를 추가하고 이미지의 경로, 이미지 타입, 사이즈를 작성한다.

```js
// layout.tsx

    <html lang="en">
      {/* title */}
      <head>
        <link rel="icon" href="/favicon.png" type="image/png" sizes="32x32" />
        <title>Tami Kim</title>
        {/* will add meta tag */}
      </head>
```

3. 새로고침 하면 아래와 같이 생성된 파비콘을 확인할 수 있다!
   ![image](https://github.com/tamoimi/tami-dev-blog/assets/100749520/2facf929-7e1d-476f-8aa7-3a6b1217f43b)
